# To-Do List App

A simple To-Do List App for adding, completing, deleting tasks with a clean, minimalistic interface

## Features

- Add new tasks
- Mark tasks as complete (with checkmark + strikethrough)
- Delete tasks
- Clean, Distraction-free UI

## Built With

- HTML
- CSS
- JavaScript

## Getting Started

1. Clone the Repository
```bash
git clone https://github.com/mhdakiv/ToDo-List-App.git
```

2. Open index.html in your browser

## Usage

- Type a task in the input field
- Click Add to add it to the list
- Click the circle next to a task to mark it complete
- Click the x to delete a task
